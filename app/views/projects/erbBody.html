The Educational Records Bureau (ERB) developed the Achievement Assessment for Beginning Learners (AABL) to replace its desktop assessment for children entering pre-k, kindergarten and first grade.

VGD created the standardized testing platform for iPad with a web-based dashboard for supervisors and administrators. I was the UX lead on the project, while supporting our dev team and project management.

Admin Dashboard

We built a web-app for test administrators to monitor student progress, take notes, and receive alerts if students need help.

The admin dashboard is designed to be usable on both tablets and desktop computers.

Usability Testing

We validated students' comfort with the ipad and the interface, but in-class testing revealed that we had overlooked a few situations.

For example, the app instructed students to sit quietly and wait between sections, but the last screen did not. So students would speak up and leave their desks, disturbing those that hadn't finished.
